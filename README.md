2.创建变量
  ACCOUNTS_JSON
  ——————————————————————————————————————————————————————————————————————————————————————————————————————————————————————————————
  {
  "accounts": [
    {
      "username": "user1",
      "password": "password1",
      "type": "ct8",
      "cronCommands": [
        "/usr/home/serv00name/domains",
        "你的脚本"
      ]
    },
    {
      "username": "user2",
      "password": "password2",
      "panelnum": "2",
      "type": "serv00",
      "cronCommands": [
        "/usr/home/serv00name"
      ]
    },
    {
      "username": "user3",
      "password": "password3",
      "type": "ct8",
      "cronCommands": [
        "/usr/home/serv00name",
        "你的脚本",
        "python /usr/home/serv00name/domains/backup.py"
      ]
    }
  ]
}

————————————————————————————————————————————————————————————————————————————————————————————————————————————————————————————————
cronCommands这部分是你想要添加的cron任务，可以是路径，也可以是程序运行的命令(一个引号内是完整的执行命令)
